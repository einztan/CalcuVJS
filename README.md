# CalcuVJS
A single Operator Calculator
It has 2 modes of Interface:
-Input Interface
-Numpad Interface

Syntaxes, Methods & Fundamentals used in this Project:
  - functions
  - callback eventListeners
  - if-else statements
  - array & object
  - for loop
  - join(), split(), includes(), push(), parseInt(), parseFloat()
  - value, length
  - JS styling & animations
